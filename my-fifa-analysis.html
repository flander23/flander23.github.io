Data Introduction

This dataset is downloaded from Kaggle for free and it shows the information of FIFA athletes. I am using this dataset to derive some basic insights about FIFA players and therefore understand better the current soccer markets.

This dataset has key variables as such:

  1. Name
  2. Nationality
  3. Age
  4. Wage (measured in Euro)

Data Visualization

Below is a histogram showing the distribution of average wage across country. By doing that we want to see the most expensive FIFA athletes across the country and so as to give the investors and the coaches some guidance in regard to hiring the most valuable FIFA soccer players. I chose to use VegaLite’s mark as ‘bar’ because a histogram is the most efficient way of describing the relationship of average wage across the country. We chose to use bar chart because it is the most efficient way to describe the relationship.

Below is another histogram showing the distribution of average age across country. We want to compare this plot with the previous plot to understand if the age is correlated with wage. Is there any positive correlation between age and wage meaning the older the soccer player the more money they make? All the information can be seen from the plot where an average wage value is calculated in each country. We chose to use bar chart because it is the most efficient way to describe the relationship.
